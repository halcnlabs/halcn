# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in halcn, please report it responsibly.

Do not open a public issue. Instead, email the maintainers directly.

## Scope

The following are in scope for security reports:

- On-chain program logic (programs/halcn-core/)
- SDK authentication and transaction signing (sdk/src/)
- PDA derivation correctness

## Response Timeline

We aim to acknowledge reports within 48 hours and provide a fix timeline within 7 days.
